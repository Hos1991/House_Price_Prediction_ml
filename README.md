# House Price Prediction ML Project

## Project Overview
This project predicts house prices using machine learning models.

## Dataset
- Source: Kaggle House Prices Dataset
- Target: SalePrice

## Steps
- Data Loading
- EDA
- Feature Engineering
- Linear Regression
- Random Forest
- Evaluation

## Features Used
- OverallQual
- GrLivArea
- TotalBsmtSF
- Age
- LotArea_log
- GarageArea
- GarageCars
- TotRmsAbvGrd
- FullBath

## Results
- Linear Regression MAE: 24501
- Random Forest MAE: 18724

## Key Insights
- OverallQual was the most important feature
- Log transformation improved LotArea
- Random Forest outperformed Linear Regression
